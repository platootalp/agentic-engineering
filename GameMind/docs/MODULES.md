### 工作流程图

```

│                         Agent Engine 工作流                               │
│                                                                            │
│    ┌─────────────────────────────────────────────────────────────────┐    │

│    └─────────────────────────────────────────────────────────────────┘    │
│                                    │                                      │
│                                    ▼                                      │
│    ┌─────────────────────────────────────────────────────────────────┐    │
│    │                     STEP 1: PLAN (规划)                         │    │
│    │  ┌───────────────────────────────────────────────────────────┐   │    │
│    │  │ 1. 加载启用的品类配置 (from DB)                         │   │    │
│    │  │ 2. 获取历史报告摘要 (避免重复)                           │   │    │
│    │  │ 3. 生成搜索查询计划 (LLM 或 keyword-based)             │   │    │
│    │  │ 4. 确定分析维度                                          │   │    │
│    │  └───────────────────────────────────────────────────────────┘   │    │
│    │  Output: ExecutionPlan { search_queries[], analysis_dimensions[]│   │
│    └─────────────────────────────────────────────────────────────────┘    │
│                                    │                                      │
│                                    ▼                                      │
│    ┌─────────────────────────────────────────────────────────────────┐    │
│    │                    STEP 2: SEARCH (搜索)                        │    │
│    │  ┌───────────────────────────────────────────────────────────┐   │    │
│    │  │                    并行执行所有数据源                      │   │    │
│    │  │                                                          │   │    │
│    │  │  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐    │   │    │

│    │  │  │  Web搜索   │  │   榜单      │  │   榜单      │    │   │    │

│    │  │                    ▼                                  │   │    │
│    │  │           SearchResults Aggregator                    │   │    │
│    │  └───────────────────────────────────────────────────────────┘   │    │
│    │  Output: SearchResults { exa[], appstore[], gp[] }           │    │
│    └─────────────────────────────────────────────────────────────────┘    │
│                                    │                                      │
│                                    ▼                                      │
│    ┌─────────────────────────────────────────────────────────────────┐    │

## 模块 6: 定时调度 (Scheduler)

### 概述
自动化报告生成的调度系统。

### 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `SCHEDULER_ENABLED` | true | 启用定时任务 |
| `SCHEDULER_HOUR` | 9 | 执行时间 (北京时间) |
| `SCHEDULER_TIMEZONE` | Asia/Shanghai | 时区 |

### 调度流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                      Scheduler 工作流程                                │
└─────────────────────────────────────────────────────────────────────┘

  APScheduler (AsyncIOScheduler)
         │
         │  CronTrigger: 0 9 * * * (每日9:00 AM 北京时间)
         │
         ▼
  ┌─────────────────┐
  │ _scheduled_exec │
  └────────┬────────┘
           │
           │ asyncio.create_task()
           │
           ▼
  ┌─────────────────┐     ┌─────────────────┐
  │ ExecutionLock   │────→│ AgentEngine     │
  │ acquire()       │     │ execute()       │
  └─────────────────┘     │ (5步流程)        │
           │              └─────────────────┘
           │                        │
           │              ┌─────────┼─────────┐
           │              │         │         │
           │              ▼         ▼         ▼
           │          ┌─────┐  ┌─────┐  ┌─────┐
           │          │Plan │  │Search│ │Analy│
           │          └─────┘  └─────┘  └─────┘
           │                                │
           │              ┌─────────────────┘
           │              ▼
           │          ┌─────┐
           │          │Report│
           │          └──┬──┘
           │             │
           │             ▼
           │  ┌──────────────────────┐
           │  │ ExecutionLock.release()│
           └→│  保存报告到数据库       │
              └──────────────────────┘
```
