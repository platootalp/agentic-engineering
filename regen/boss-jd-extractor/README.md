# Boss直聘JD提取器 v2.0

一键提取Boss直聘岗位JD，自动异步提取并下载Markdown文件。

## 功能特点

- ✅ **一键提取** - 单个按钮，自动检测页面类型
- ✅ **智能识别** - 自动区分职位列表页和详情页
- ✅ **后台异步** - 列表页自动打开每个职位详情页提取完整JD
- ✅ **进度显示** - 实时显示提取进度
- ✅ **自动下载** - 提取完成后一键下载Markdown文件
- ✅ **断点续传** - 支持取消和重新提取

## 安装步骤

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 右上角打开「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `boss-jd-extractor` 文件夹

## 使用方法

### 1. 职位列表页提取（推荐）

1. 打开Boss直聘职位列表页：`https://www.zhipin.com/web/geek/jobs`
2. 点击浏览器工具栏的插件图标
3. 点击「开始提取」按钮
4. 等待提取完成（插件会自动打开每个职位详情页提取完整JD）
5. 点击「下载Markdown」保存文件

### 2. 职位详情页提取

1. 打开任意职位详情页
2. 点击插件图标
3. 点击「开始提取」
4. 提取完成后下载Markdown

## 提取内容

- 📌 职位名称
- 💰 薪资范围
- 🏢 公司名称
- 📍 工作地点
- 👤 经验要求
- 🎓 学历要求
- 🏷️ 技能标签
- 📝 完整职位描述
- 👔 招聘官信息
- 🔗 原始链接

## 文件结构

```
boss-jd-extractor/
├── manifest.json      # 插件配置
├── background.js      # 后台服务（处理批量提取）
├── content.js         # 页面内容提取脚本
├── popup.html         # 弹出界面
├── popup.js           # 弹出界面逻辑
└── icons/             # 图标文件
```

## 注意事项

1. **提取时间** - 列表页提取需要较长时间（每个职位约3-5秒）
2. **保持页面打开** - 提取过程中请勿关闭浏览器
3. **可取消** - 提取过程中可随时点击「取消提取」
4. **自动保存** - 提取结果会自动保存，关闭插件后仍可下载
